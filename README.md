# Git

<b>Force merge unsynced branch to main</b>
<br/>
--allow-unrelated-histories

<b>Refresh .gitignore</b>
<br/>
1. Make changes in .gitignore file.
2. git rm -r --cached .
3. git add .
4. git commit -m "Commit message"
